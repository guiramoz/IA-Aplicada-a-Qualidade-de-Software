# IA-Aplicada-ao-Desenvolvimento-de-Software
Atividade desenvolvida no curso de Inteligências Artificiais Generativas Aplicada ao ChatGPT do SENAI. Explora o uso de Engenharia de Prompt, Inteligência Artificial para correção de erros de código e criação de documentações técnicas relacionadas a bugs, melhorias e novas funcionalidades em projetos de software.
